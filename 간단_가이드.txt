📱 휴대폰에서 테스트하는 방법 (간단 버전)

✅ 개발 서버는 이미 실행 중입니다!

다음 단계만 하면 됩니다:

1. 새 터미널 창을 엽니다 (PowerShell 또는 CMD)

2. 다음 명령어를 입력하세요:
   npx ngrok http 3000

3. 화면에 나타나는 HTTPS URL을 복사하세요
   예: https://xxxx-xxxx-xxxx.ngrok-free.app

4. 휴대폰 브라우저에서 그 URL로 접속하세요!

5. 또는 ngrok 웹 인터페이스 확인:
   컴퓨터 브라우저에서 http://127.0.0.1:4040 접속
   → "Forwarding" 섹션에서 HTTPS URL 확인

⚠️ 중요:
- ngrok을 실행한 터미널은 닫지 마세요!
- ngrok URL은 매번 바뀝니다
- 무료 버전은 세션당 2시간 제한이 있습니다

💡 더 안정적인 방법:
Vercel에 배포하면 고정된 HTTPS URL을 사용할 수 있습니다!
git push만 하면 자동으로 배포됩니다.
